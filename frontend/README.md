# Cent Stage - Frontend

This is the Next.js + Tailwind frontend for the Cent Stage task manager.

Quick start

```bash
cd frontend
npm install
cp .env.example .env.local
# edit .env.local if needed
npm run dev
```

By default the frontend calls `http://localhost:4000` for the backend. Change `NEXT_PUBLIC_API_URL` in `.env.local` to point to your backend if different.
